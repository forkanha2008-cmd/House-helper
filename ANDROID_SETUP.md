# House Helper — Android Build Guide

## Project structure after setup

```
house-helper-pwa-v2/          ← project root (run all commands here)
├── www/                      ← webDir — all web assets live here
│   ├── index.html            ← customer app
│   ├── admin.html            ← admin dashboard
│   ├── app.js
│   ├── style.css
│   ├── native.js
│   ├── analytics.js
│   ├── sw.js
│   ├── manifest.json
│   ├── logo.jpg
│   ├── icon-192.png
│   ├── icon-512.png
│   └── assets/
│       └── hero-cleaner.png
├── android/                  ← generated by Capacitor (do not edit manually)
├── capacitor.config.json     ← "webDir": "www"
├── package.json
├── analytics-schema.sql
└── ANDROID_SETUP.md
```

**Rule: every HTML/JS/CSS change must go into `www/`, not the project root.**

## Prerequisites

Install these before running any command:

| Tool | Version | Link |
|------|---------|------|
| Node.js | 18 or 20 LTS | https://nodejs.org |
| Java JDK | 17 (not 21) | https://adoptium.net |
| Android Studio | Latest stable | https://developer.android.com/studio |

After installing Android Studio, open it once and let it finish setup.
Then install SDK: Tools → SDK Manager → SDK Platforms → Android 14 (API 34) → Apply.

## Exact commands — run in order

Open a terminal inside the `house-helper-pwa-v2` folder, then:

### 1. Install Capacitor
```bash
npm install
```

### 2. Add the Android platform
```bash
npx cap add android
```
This creates the `android/` folder. Takes 1–2 minutes.

### 3. Sync web assets to Android
```bash
npx cap sync android
```
This copies everything in `www/` into `android/app/src/main/assets/public/`
and generates `capacitor.plugins.json`. You must run this every time you
change any file in `www/`.

### 4. Open in Android Studio
```bash
npx cap open android
```
Wait for Gradle sync to finish (first time: 5–10 minutes, downloads dependencies).

### 5. Run on device or emulator
In Android Studio: click the green ▶ Run button, select your device.

## Add required Android permissions

Open `android/app/src/main/AndroidManifest.xml`.
Add these lines inside `<manifest>` before `<application>`:

```xml
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.CAMERA" />
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"
    android:maxSdkVersion="32" />
<uses-permission android:name="android.permission.VIBRATE" />
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
```

## Replace app icon

1. Go to `android/app/src/main/res/`
2. Replace `ic_launcher.png` in each folder with your icon:

| Folder | Size |
|--------|------|
| `mipmap-mdpi` | 48×48 |
| `mipmap-hdpi` | 72×72 |
| `mipmap-xhdpi` | 96×96 |
| `mipmap-xxhdpi` | 144×144 |
| `mipmap-xxxhdpi` | 192×192 |

Or use Android Studio shortcut:
Right-click `res` folder → New → Image Asset → select your logo.

---

## Add splash screen

1. Copy your splash image to:
   `android/app/src/main/res/drawable/splash.png`
2. Recommended size: 2048×2048 px (white or `#0F172A` dark background).

## Build a release APK

### Step 1 — Create a keystore (do once, save the file securely)
```bash
keytool -genkey -v \
  -keystore house-helper.keystore \
  -alias house-helper \
  -keyalg RSA \
  -keysize 2048 \
  -validity 10000
```
Remember the password you set.

### Step 2 — Build the signed APK
```bash
cd android
./gradlew assembleRelease
```

APK is at:
`android/app/build/outputs/apk/release/app-release.apk`

### Step 3 — Sign the APK manually (if unsigned)
```bash
jarsigner -verbose \
  -keystore ../house-helper.keystore \
  android/app/build/outputs/apk/release/app-release-unsigned.apk \
  house-helper
```

## After changing web files

Whenever you edit anything in `www/`:
```bash
npx cap sync android
```
Then rebuild in Android Studio.

## Troubleshooting

**Error: webDir is not valid**
→ Make sure `capacitor.config.json` has `"webDir": "www"` (not `"."`)
→ Make sure the `www/` folder exists and contains `index.html`

**Error: capacitor.plugins.json not found**
→ Run `npx cap sync android` — this generates that file automatically
→ It only works after `npx cap add android` has been run first

**Error: SDK location not found**
→ In Android Studio: File → Project Structure → SDK Location → set Android SDK path

**Gradle sync failed**
→ Make sure Java 17 is installed: `java -version`
→ File → Invalidate Caches → Restart

**Build failed: duplicate class**
```bash
cd android && ./gradlew clean
```
Then sync and build again.

**App opens but shows blank white screen**
→ `www/index.html` is missing or `npx cap sync android` was not run
→ Run `npx cap sync android` and rebuild

**Images not uploading from device**
→ Check Android permissions in AndroidManifest.xml
→ Check Supabase RLS policies (run `fix-storage-rls.sql`)
